Available Endpoints

Collections

  • GET /collection/ - Return the list of registered collections (documentation)
  • PUT /collection/ - Update list of registered collections (documentation)
  • POST /collection/ - Register a new collection or update metadata of a registered collection (documentation)
  • GET /collection/:id - Return metadata of a specific registered collection (documentation)
  • PUT /collection/:id - Update metadata of a specific registered collection or register a new collection (documentation)
  • DELETE /collection/:id - Unregister a collection and remove it from the knowledge graph (documentation)
  • POST /collection/:id/receive - Receive and process collection data (documentation)
  • GET /collection/:id/receive - Get latest receive log of a collection (documentation)
  • POST /collection/:id/load - Load received and processed collection data into the triple store (documentation)
  • GET /collection/:id/load - Get latest load log of a collection (documentation)
  • POST /collection/:id/remove - Remove collection data and metadata from the knowledge graph and from staging area (documentation)
  • GET /collection/:id/remove - Get latest remove log of a collection (documentation)

Terminologies

  • GET /terminology/ - Return the list of registered terminology metadata (documentation)
  • GET /terminology/:id - Return metadata of a registered terminology (documentation)
  • PUT /terminology/:id - Add or update metadata of a registered terminology from BARTOC (documentation)
  • POST /terminology/:id/receive - Receive terminology data (documentation)
  • GET /terminology/:id/receive - Get latest receive log of a terminology (documentation)
  • POST /terminology/:id/load - Import terminology data into the triple store (documentation)
  • GET /terminology/:id/load - Get latest load log of a terminology (documentation)
