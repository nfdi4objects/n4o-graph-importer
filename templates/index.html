Available Endpoints

  • GET /initCT - Initializes list of collections and terminologies (this will be removed)

Collections

  • GET /collection/ - Returns list of registered collections (documentation)
  • GET /collection/:id - Returns metadata of a specific registered collection (documentation)
  • PUT /collection/ - Update list of registered collections (documentation)
  • PUT /collection/:id - Updates metadata of a specific registered collection (documentation)
  • POST /collection - Register a new collection (documentation)
  • DELETE /collection/:id - Unregister a collection and remove it from the knowledge graph (documentation)
  • GET /collection/:id/:file - Returns a file from the staging area of a registered collection (documentation)
  • POST /collection/:id/receive - Receive and process collection data (documentation)
  • POST /collection/:id/import - Load received and processed collection data into the triple store (documentation)
  • DELETE /collection/:id/delete - Delete collection data from the knowledge graph and from staging area (documentation)
